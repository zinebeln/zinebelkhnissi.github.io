MOBILE - ARDUINO Mobile developpement

Découvrez l'essence de notre projet NeoPixel sur ESP32, alimenté par Arduino, où le défilement dynamique des mots prend vie. Cette initiative exploite une matrice de NeoPixels pour afficher les mots envoyés depuis une page Web au sein du même réseau. Les utilisateurs peuvent régler la luminosité des LED via un potentiomètre. L'interface de la page Web du projet permet une transmission transparente des mots à afficher.

JAVA - NETBEANS APPLICATION BUREAU DEVELOPPEMENT

Plongez dans le jeu classique de 2048, réinventé grâce à mon projet codé en Java. Cette application donne vie au jeu de puzzle, en mettant les joueurs au défi de faire glisser des tuiles numérotées sur une grille pour les combiner en de plus grands nombres, dans le but ultime d'atteindre la tuile 2048.

KOTLIN - ANDROID MOBILE DEVELOPPEMENT

Plongez dans le domaine numérique avec mon Hearthstone Manager, une application mobile innovante conçue en Kotlin dans une architecture MVVM. Cette application intuitive révolutionne la façon dont les joueurs de Hearthstone interagissent avec leurs cartes préférées, offrant une connexion transparente, la possibilité d'ajouter des cartes à une liste de favoris et une personnalisation de profil.

BOOSTRAP - NODE JS - JAVASCRIPT WEB DEVELOPPEMENT

Embarquez pour une aventure en ligne avec notre jeu Web "Qwixx", un jeu de dés complexe codé en Java, Bootstrap et Node.js, pouvant accueillir de 2 à 10 joueurs. Cette application, hébergée sur un serveur Web, permet aux joueurs d'élaborer des stratégies et de marquer des numéros sur quatre lignes colorées sur leur feuille de score, dans le but d'accumuler le plus de points. Tout cela est soutenu par une base de données back-end capturant les données et les scores des joueurs, faisant de chaque jeu un défi unique dans la quête de l'excellence stratégique.

ESP32 - FIREBASE MOBILE DEVELOPPEMENT

Découvrez la fusion de l'IoT avec mon projet intégré Arduino et Firebase, un système d'alarme avancé de détection de mouvement. Cette solution de pointe déclenche une alarme lors de la détection d'un mouvement, tout en offrant la possibilité de la désactiver via une application mobile liée. De plus, les utilisateurs peuvent accéder à l'application pour consulter l'historique de leurs actions, garantissant ainsi une compréhension complète de l'activité et de la sécurité du système à portée de main.
